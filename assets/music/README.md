# Background Music Tracks

Drop royalty-free instrumental MP3s here. The pipeline auto-selects the best match.

## Naming convention
Name files to describe the mood/ending type. Examples:
- `sad_piano_rain.mp3`
- `bittersweet_strings_hope.mp3`
- `happy_warm_acoustic.mp3`
- `tense_cinematic_dark.mp3`
- `melancholic_cello_night.mp3`

## Free sources (no attribution required)
- **Pixabay Music** — pixabay.com/music — filter by "cinematic", "sad", "inspirational"
- **Free Music Archive** — freemusicarchive.org — CC0 or CC-BY tracks
- **Incompetech** — incompetech.com/music/royalty-free — Kevin MacLeod, CC-BY 4.0
- **Musopen** — musopen.org — public domain classical recordings
- **ccMixter** — dig.ccmixter.org — filter by CC0
